# Console-Based Music Player (Java)

This Mini project is developed using Java and Object-Oriented Programming concepts.

## Features

* Add, update, and delete songs
* Create and manage playlists
* Add songs to playlists
* Play, pause, and stop songs
* Menu-driven console base application

## Concepts Used

* Encapsulation
* Abstraction (Interface)
* Inheritance
* Polymorphism
* Collections (ArrayList)
